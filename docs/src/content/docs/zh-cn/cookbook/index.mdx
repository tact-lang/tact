制作 Tact Cookbook 的主要目的是将 Tact 开发人员的所有经验汇集到一个地方，以便未来的开发人员可以使用。 这部分文档更侧重于每个 Tact 开发人员在开发智能合约过程中需要解决的日常任务。

将其作为在 TON 区块链上制作令人愉悦的智能合约的食谱，而无需在此过程中重新发明轮子。
