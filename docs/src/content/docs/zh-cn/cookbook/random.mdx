## 如何生成随机数

```tact
// 声明一个变量来存储随机数
let number: Int;

// 生成一个新的随机数，它是一个无符号的 256 位整数
number = randomInt();

// 生成一个介于 1 和 12 之间的随机数
number = random(1, 12)；
```
